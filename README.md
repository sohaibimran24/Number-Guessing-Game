# 🎯 Number Guessing Game

A fun number guessing game built with Python.

## 🚀 About The Project
The computer picks a random number and you have to guess it! 
The game gives you hints whether your guess is too high or too low.

## ✨ Features
- 3 difficulty levels (Easy, Medium, Hard)
- Limited attempts based on difficulty
- Hints after every guess
- Score tracking
- Play again option

## 🛠️ Built With
- Python 3
